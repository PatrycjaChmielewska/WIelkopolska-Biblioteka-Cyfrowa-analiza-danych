#  Wielkopolska Biblioteka Cyfrowa (Great Poland Digital Library) - Data Analysis

This repository contains a data analysis project focusing on the statistics of the Wielkopolska Biblioteka Cyfrowa. 

## Project Objectives
The main goal of this analysis was to process library data and answer four key questions:
1. What is the average number of objects added per month between 2004 and 2025? 
2. What are the most frequently used data formats? 
3. What percentage of the data objects has been shared? 
4. What are the top 100 most frequently viewed types of objects? 

##  Tools & Technologies Used
- **Language:** R 
- **IDE:** RStudio 
- **Libraries:** - `readr` for data import 
  - `ggplot2` for creating data visualizations 
  - `dplyr` for data manipulation 

##  Key Findings
- **Format Dominance:** The DJVU format is overwhelmingly the most popular format for shared objects, followed by PDF and photo galleries.
- **Categories:** A significant portion of the shared objects includes magazines ("czasopismo") and newspapers ("gazeta"). 
- **Top Views:** Among the most viewed categories are doctoral dissertations ("rozprawa doktorska") with nearly 9.7 million views, and magazines ("czasopisma") with nearly 8 million views.
- **Statistical Metrics:** Calculated metrics for view counts revealed an average of 792,865 and a median of 271,081.5.

*Note: The complete methodology, scripts, and visualizations (including bar charts and pie charts) are available in the attached PDF presentation.*
